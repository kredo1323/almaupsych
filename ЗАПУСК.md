# 🌿 AlmauPsych — Запуск сайта

## Требования
- [Node.js](https://nodejs.org) (скачать и установить, если нет)

---

## Первый запуск

1. Удалите папку `node_modules` (если есть)
2. Откройте эту папку в PowerShell или CMD:
   ```
   cd "путь\к\сайт психологинь"
   ```
3. Установите зависимости:
   ```
   npm install
   ```
4. Запустите сервер:
   ```
   node server.js
   ```
5. Откройте браузер: **http://localhost:3000**

---

## Следующие запуски

Просто запустите:
```
node server.js
```
